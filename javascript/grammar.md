# 文法

各行はセミコロンを文末に置くことで区別する。
1 行で完結する場合は書かなくていいが、基本的に書くことを推奨する。

## コメントアウト

コメントアウトは以下。

```js
// 一行ならこれ
/*
これで複数
行もいける
*/
```

## 宣言

`var`: 変数を宣言し、初期化もできる。非推奨されているらしい。
`let`: グローバル、またはブロックスコープ内部で変数を宣言し、初期化もできる。
`const`: 定数の宣言。再代入は出来ない。

let と const は変数と定数という違いはあるものの、スコープについては挙動が同じ

定数 const は、再代入が禁止されているだけであって、変更はできる。

```js
const MY_OBJECT = { key: "value1" };
MY_OBJECT.key = "Other value";
```

## データ型

数値や文字列関係なく、**動的型付け言語**であるため、let で変更ができる。Python と同じ。
以下のようなこともできる。

```js
let x = 42;
x = "Hello world!";
```

## リスト

空白の要素がある場合、これもカウントされてしまうのが注意事項。
たとえば

```js
const drinks = ["coffee", , "water", "tea"];
console.log(drinks);
// (4) ['coffee', empty, 'water', 'tea']
```

## 辞書型(オブジェクトリテラル)

```js
const car = { manyCars: { a: "サーブ", b: "ジープ" }, 7: "マツダ" };

console.log(car.manyCars.b); // ジープ
console.log(car[7]); // マツダ
```

`[]`で key を呼んでもいいし、`.key`で呼んでもいい。
