# if 文、比較演算子、論理演算子

```cpp
if (条件式1) {
  処理1
}
else if (条件式2) {
  処理2
}
else {
  処理3
}

```

演算子は以下の通り。
|演算子|意味|
|-|-|
|`x==y`|x と y は等しい|
|`x!=y`|x と y は等しくない|
|`x>y`|x は y より大きい|
|`x<y`|x は y より小さい|
|`x>=y`|x は y 以上|
|`x<=y`|x は y 以下|
