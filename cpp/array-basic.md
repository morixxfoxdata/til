# 配列

## 配列の宣言

`vector<型> 配列変数名(要素数)`で定義できる。
要素数は指定しなくてもいい。

```cpp
vector<int> vec(N);
```

## 配列変数への代入

`配列変数 = {要素1, 要素2, ...};`

## i 番目の抜き出し

`vec.at(i)`で文字列同様抜き出しが可能。
`vec.size()`で要素数を取得できる。

## 初期値設定

配列初期化時の初期値を指定できる。
`vector<型> 配列名(要素数, 初期値);`

## 配列への要素追加

文字列のように`+=`で要素を追加できない。その代わり`配列変数.push_back`で末尾に追加できる。

## 配列の末尾要素削除

`配列変数.pop_back`で末尾の要素を削除できる。

## vector 以外の配列

```cpp
vector<int> data(3); // vectorによる配列
int data[3]; // Cの配列
array<int, 3> data; // arrayによる配列

```
