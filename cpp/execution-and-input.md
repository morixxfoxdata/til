# 実行順序と入力

基本的にプログラムは上から下へ順番に実行される。

## 入力

入力を受け取る場合は`cin >>`を使用する。

以下は受け取った数値を 10 倍にして出力するプログラム。

```cpp
int main() {
    int a;

    cin >> a;

    cout << a * 10 << endl;
}
```

整数以外のデータを受け取りたい場合は受け取りたいデータ型に合わせた変数宣言を行う。

`cout`同様に`cin`も繋げて受け取ることができる。
複数入力がある場合、スペースか改行で区切られていれば分解して入力してくれる。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int a, b, c;

  cin >> a >> b >> c;
  cout << a * b * c << endl;
}
```

入力

```
2 3
4
```

出力

```
24
```
