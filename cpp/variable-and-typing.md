# 変数と型

変数は「メモ」である。
一度名前をつけておいて、後で使い回す。

変数を使うには**宣言**をする必要がある。
宣言時にはデータの型と変数の名前を指定する。

```cpp
int name;
```

| 型           | 内容       |
| ------------ | ---------- |
| int          | 整数       |
| double       | 小数       |
| 　 string 　 | 　文字列　 |

宣言された変数へ値を代入するには`=`を使用する。

```cpp
name = 10;
```

`=`は代入であって、等しいという意味ではないことに留意する。

## 変数の初期化

変数の宣言と代入は同時に行うこともできる。

```cpp
int name = 10;
```

## 変数のコピー

`変数1=変数2`と書いた場合、変数の値そのものがコピーされる。
その後にどちらかの変数の値が変更されても、もう片方の変数は影響を受けない。

```cpp
int main() {
    int a = 10;
    int b;

    b = a;
    a = 5;

    cout << a << endl;
    cout << b << endl;
}
```

```
5
10
```

変数はカンマで区切って複数同時に宣言可能。

```cpp
int a = 4, b = 5;
```

## 変数名のルール

- 数字で始まる名前
- `_`以外の記号が使われている名前
- キーワード(C++が使っている単語の一部)

```cpp
int 100hello; // 数字で始まる名前にはできない
int na+me; // 変数名に+を使うことはできない
int int; // intはキーワードなので変数名にできない
```

## 異なる型同士の計算

int 型と double 型のように異なる型で計算する場合は型変換が行われ、double 型の計算結果が得られる。
string と int、string と double などの変換できない型の組み合わせは当然エラーになる。

## 異なる型への代入

```cpp
int i = 1;
string a = "Good";
i = a; // int型に対してstring型は代入できない
```
